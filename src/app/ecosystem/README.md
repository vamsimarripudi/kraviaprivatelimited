# src/app/ecosystem

Phase 17 frontend module boundary for the Ecosystem Control Plane.

The current app is a vanilla single-file workspace, so the live rendering is implemented in `app.js` while this folder documents the module boundary for a future split into framework modules. The active UI section exposes:
- Ecosystem Dashboard
- Product Registry
- Product Health
- Product Revenue Summary
- Product Roadmap
- Product Launch Checklist
- Product Risk Register
- Ecosystem Control Plane Report
