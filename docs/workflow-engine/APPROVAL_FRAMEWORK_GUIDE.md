# KRAVIA Company OS - Approval Framework Guide

## Supported Approval Modes

- Single approval
- Sequential approval
- Parallel approval
- Multi-level approval
- Quorum approval
- Conditional approval

## Configuration

Approval behavior is configured through workflow template steps.

Each step may define:

- Step type
- Approval mode
- Approver role
- Approver user
- Escalation window
- Completion rule

## Permissions

Founder:

- Creates and manages workflow templates, rules, and schedules.

Director:

- Starts and manages operational workflow execution.

Viewer:

- Views workflow status only.
