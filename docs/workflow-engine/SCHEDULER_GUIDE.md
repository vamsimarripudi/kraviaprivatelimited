# KRAVIA Company OS - Scheduler Guide

## Supported Frequencies

- Daily
- Weekly
- Monthly
- Quarterly
- Annually
- Cron

## Scheduled Job Fields

- Job name
- Job key
- Frequency
- Cron expression
- Next run time
- Last run time
- Status
- Action summary

## Example Uses

- Daily compliance review
- Weekly board summary
- Monthly finance review
- Quarterly access review
- Annual compliance reminder

## Statuses

- Active
- Paused
- Failed
- Archived

## Audit

Creating, updating, and archiving scheduled jobs creates audit logs.
