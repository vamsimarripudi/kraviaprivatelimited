# Vendor Management Guide

## Purpose
Use Vendor Master to keep a controlled list of KRAVIA vendors, service providers, banks, consultants, and recurring suppliers.

## Add A Vendor
1. Open Procurement.
2. Select Vendor Master.
3. Enter vendor name and category.
4. Add contact, tax, service, address, and notes when available.
5. Save the record.

## Vendor Data Rules
- Do not add informal or unverified vendor names.
- Add GSTIN and PAN only when available from verified documents.
- Use notes for internal context, not sensitive passwords or secrets.
- Archive inactive vendors instead of deleting records permanently.

## Vendor Documents
Vendor-related contracts, KYC records, invoices, or bank documents should be uploaded to Document Vault first, then linked from Vendor Documents.
