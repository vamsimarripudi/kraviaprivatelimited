# Subscription Control Guide

## Purpose
Use Subscription Tracking for recurring software, cloud, compliance, and operational subscriptions.

## Add A Subscription
1. Open Procurement.
2. Select Subscriptions.
3. Add service name, vendor, plan, billing cycle, amount, renewal date, owner, and status.
4. Mark auto-renewal accurately.

## Renewal Review
Review upcoming renewals before payment or renewal date.

Recommended checks:
- Is the service still used?
- Is the plan still appropriate?
- Is there a better annual or monthly billing option?
- Is the owner still responsible?
- Is payment status reflected in Finance ERP where applicable?

## Security
Do not store API keys, passwords, or card details in subscription notes.
