# Assignment Workflow

## Assign An Asset
1. Open Asset Management.
2. Select Assignments.
3. Choose the asset.
4. Enter the assigned person, assigned date, location, and status.
5. Save the assignment.

## System Behavior
When an assignment is marked assigned, the linked asset is updated to assigned and stores the assigned person and location.

When an assignment has a return date or is marked unassigned, the linked asset is moved back to unassigned.

## Controls
- Founder and Director can assign assets.
- Viewer can only view assignment history.
- Assignment changes are written to audit logs.
