# Maintenance Workflow

## Add Maintenance Record
1. Open Asset Management.
2. Select Maintenance.
3. Choose the asset.
4. Enter maintenance title, service provider, date, next maintenance date, cost, status, and notes.
5. Save the maintenance record.

## Maintenance Rules
- Use maintenance records for repairs, warranty service, inspections, renewals, or infrastructure checks.
- Do not store passwords, API keys, or secrets in notes.
- Mark assets under maintenance only when the asset is actually unavailable or being serviced.

## Audit Trail
Every maintenance create, update, status change, and archive action is recorded in audit logs.
