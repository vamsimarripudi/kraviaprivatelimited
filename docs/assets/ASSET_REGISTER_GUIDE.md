# Asset Register Guide

## Purpose
Use Asset Register to track KRAVIA physical, digital, software, cloud, and infrastructure assets.

## Add An Asset
1. Open Asset Management.
2. Select Asset Register.
3. Enter asset name and asset code.
4. Choose the category and status.
5. Add purchase, vendor, warranty, renewal, location, and document references when available.
6. Save the record.

## Data Rules
- Asset codes must be unique.
- Do not create placeholder assets.
- Link vendors only from Procurement Vendor Master.
- Link documents only after the file exists in Document Vault.
- Archive retired records instead of removing traceability.

## Recommended Asset Codes
Use a consistent internal naming pattern decided by KRAVIA leadership. Keep codes stable after assignment.
