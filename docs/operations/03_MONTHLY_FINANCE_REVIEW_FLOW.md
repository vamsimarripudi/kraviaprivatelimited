# Monthly Finance Review Flow

## Purpose

Use this flow to record and review KRAVIA monthly financial position.

This is a founder-level summary. It is not a replacement for accounting software or CA records.

## Recommended Owner

Founder or Director, with input from CA or accountant where required.

## Before You Start

Collect source records:

- Bank statements
- Invoices
- Receipts
- Payment gateway summaries
- GST summaries
- Vendor bills
- Cloud and software bills
- Director remuneration records, if applicable

## Monthly Review Steps

1. Open Financial Records.
2. Search for the reporting month.
3. If a record exists, open and review it.
4. If no record exists, create a new monthly financial record.
5. Enter revenue.
6. Enter expenses.
7. Enter cash balance.
8. Enter receivables and payables.
9. Enter GST collected and GST paid.
10. Enter cloud subscriptions, vendor payments, and director remuneration if available.
11. Review system-calculated profit/loss.
12. Review system-calculated net GST position.
13. Add founder notes for important context.
14. Save the record.

## Review Checks

Confirm:

- Reporting month is correct.
- Currency values come from source records.
- Zero values are intentional.
- Negative values are explained where needed.
- GST position matches available tax records.
- Notes explain unusual changes.

## After Saving

1. Generate a Financial Summary Report if needed.
2. Share with CA or directors through approved channels.
3. Create tasks for missing documents or follow-up work.
4. Upload supporting documents to Document Vault if they belong in company records.

## Finance Rule

Do not enter guesses as official records. If a value is not available, leave it empty where allowed or record the limitation in notes.
