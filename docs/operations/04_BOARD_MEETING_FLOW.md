# Board Meeting Flow

## Purpose

Use this flow to record board meetings, founder meetings, finance reviews, product reviews, compliance reviews, bank meetings, and investor meetings.

## Before The Meeting

1. Open Board Meetings.
2. Create a meeting record.
3. Add meeting title.
4. Add meeting date.
5. Select meeting type.
6. Add attendees.
7. Add agenda items.
8. Save the draft.

## During The Meeting

Record:

- Discussion notes
- Decisions taken
- Board resolutions
- Action items
- Action owner
- Due date
- Next meeting date, if known

Keep notes factual. Do not include informal comments that should not become company records.

## After The Meeting

1. Review the meeting record.
2. Confirm every decision is clear.
3. Confirm every action item has an owner.
4. Confirm active action items have due dates.
5. Attach related document references where available.
6. Mark status as completed when the record is complete.
7. Print or export later if a formal copy is needed.

## Action Item Rules

Every action item should have:

- Clear task
- Owner
- Due date
- Status

If the action creates operational work, create a separate task in Tasks.

## Resolution Rules

Board resolutions should be written clearly and professionally.

If legal wording is required, confirm with a company secretary, lawyer, or CA before treating it as final.
