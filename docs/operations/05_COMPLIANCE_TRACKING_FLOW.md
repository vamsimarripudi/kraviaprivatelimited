# Compliance Tracking Flow

## Purpose

Use this flow to track KRAVIA compliance obligations and avoid missed deadlines.

## Recommended Owner

Founder, Director, CA, or assigned responsible person.

## Creating A Compliance Item

1. Open Compliance Center.
2. Search first to avoid duplicates.
3. Select Add Compliance Item.
4. Enter title.
5. Select category.
6. Enter due date unless the item is not applicable.
7. Select priority.
8. Assign responsible person.
9. Add notes if needed.
10. Save.

## Updating Compliance Status

1. Open the compliance item.
2. Review current status.
3. Update status based on real progress.
4. Add notes for submission, approval, rejection, or delay.
5. Link related document if available.
6. Save.

## Due Date Review

Review regularly:

- Overdue items
- Items due soon
- High and critical priority items
- Items waiting for CA
- Items waiting for director action

## When A Compliance Item Is Complete

1. Confirm work is complete.
2. Upload supporting document if available.
3. Link the document to the compliance item where possible.
4. Set status to completed or approved.
5. Keep notes short and factual.

## Compliance Rules

- Do not mark submitted unless submission actually happened.
- Do not mark approved unless approval is confirmed.
- Do not delete records for completed compliance work.
- Archive only when the item should remain as history and no longer appears as active work.
