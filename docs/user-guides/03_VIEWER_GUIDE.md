# Viewer Guide

## Purpose

This guide explains how Viewer users should use KRAVIA Company OS.

Viewer access is read-only. It is meant for people who need to review information without changing company records.

## What Viewers Can Do

Viewers can usually:

- Log in.
- View permitted company records.
- Search permitted records.
- View permitted reports.
- Download permitted documents.
- Read announcements and notifications.

Viewers cannot:

- Create records.
- Edit records.
- Archive records.
- Upload documents.
- Complete tasks.
- Use the Executive AI Assistant unless access is later granted by policy.

## Daily Use

1. Log in with your viewer account.
2. Open the module you need to review.
3. Use search and filters to find records.
4. Open details to read the full record.
5. Download documents only when needed for approved work.
6. Log out when finished.

## Reviewing Company Profile

1. Open Company Profile.
2. Review available company details.
3. If something appears incorrect, report it to a founder or director.

Do not try to keep a separate private copy as the main record.

## Reviewing Documents

1. Open Documents.
2. Search by title or category.
3. Open metadata before downloading.
4. Download only the file needed.
5. Store downloaded documents securely.

## Reviewing Reports

1. Open Reports.
2. Choose the report type.
3. Apply date filters if needed.
4. Review the report.
5. Print only when required.

## Viewer Rules

- Do not share downloaded documents without approval.
- Do not ask another user to make edits under their account.
- Do not use screenshots as official records.
- Report incorrect or missing data to a founder or director.
- Log out on shared or borrowed devices.
