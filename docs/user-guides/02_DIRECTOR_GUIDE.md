# Director Guide

## Purpose

This guide explains how KRAVIA directors should use KRAVIA Company OS for operational company work.

Director access allows broad company work, but some final control actions remain founder-only.

## What Directors Can Do

Directors can usually:

- View company records.
- Edit Company Profile.
- Upload, view, download, and edit documents.
- Create and edit board meetings.
- Create and edit financial records.
- Create and edit compliance items.
- Create, edit, and complete tasks.
- Create and edit products and contacts.
- View permitted announcements, notifications, reports, search results, and audit logs.

Directors may not be allowed to archive some records depending on the module.

## Daily Start

1. Log in with your director account.
2. Review notifications.
3. Check tasks assigned to you.
4. Review compliance items waiting for director action.
5. Update records where work has changed.

## Editing Records

Before editing:

1. Confirm the record is the correct one.
2. Check whether another user has already updated it.
3. Enter only verified information.
4. Save changes.
5. Confirm the status now matches the real situation.

## Meetings

Directors can create and maintain meeting records.

Steps:

1. Add the meeting title, date, type, and attendees.
2. Add agenda items.
3. Record discussion notes clearly.
4. Record decisions and resolutions separately.
5. Add action items with owner, due date, and status.

Keep meeting records factual and professional.

## Finance

Directors can add and edit financial summaries where permitted.

Steps:

1. Use source records such as invoices, bank statements, and accounting summaries.
2. Enter revenue, expenses, GST collected, and GST paid.
3. Review calculated profit/loss and net GST position.
4. Add notes for unusual changes.
5. Save the record.

Do not enter numbers without source records.

## Documents

Directors can upload and manage company documents.

Steps:

1. Search for the document first.
2. Upload only final or clearly named draft documents.
3. Select the correct category.
4. Enter a clear title.
5. Keep descriptions short and useful.

## Compliance and Tasks

Steps:

1. Review active compliance items.
2. Update status when a filing, review, or approval moves forward.
3. Create tasks when another person must act.
4. Mark tasks done only when the work is complete.

## Director Rules

- Do not archive records unless the system permits it and the founder has approved it.
- Do not add placeholder information.
- Do not share confidential records outside approved channels.
- Ask the founder before changing sensitive company profile details.
- Use comments and notes to explain important updates.
