# Organization Structure Documentation

## Organization Levels

KRAVIA Company OS supports the following levels:
- Founder
- Board of Directors
- Executive Leadership
- Department Heads
- Team Leads
- Employees
- Interns
- Contractors
- Advisors

## Departments

Departments define the operating structure of the company. Each department may include:
- Department name
- Parent department
- Organization level
- Department head
- Status
- Description

## Designations

Designations define formal roles within departments. Each designation may include:
- Title
- Department
- Organization level
- Status
- Description

## Usage Guidance

Create departments before adding employee records. Add designations next, then assign employees to departments and designations.
