# Leave Workflow Guide

## Leave Request Statuses

- Requested
- Manager Review
- Approved
- Rejected
- Cancelled
- Archived

## Workflow

1. Create a leave request with employee, leave type, start date, end date, and total days.
2. Assign a manager when manager review is required.
3. Update status after review.
4. Add approval notes for accountability.
5. Archive only when the record should no longer appear in active views.

## Controls

Founder and Director roles may create and update leave requests. Viewer role can only read leave records.
