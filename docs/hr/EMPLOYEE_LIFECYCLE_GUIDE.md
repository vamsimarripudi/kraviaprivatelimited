# Employee Lifecycle Guide

## 1. Create Employee Record

Add the employee ID, full name, email, employment type, joining date, probation status, and employment status.

## 2. Assign Organization Details

Assign department, designation, reporting manager, and work location.

## 3. Maintain Employment Details

Keep skills, certifications, contacts, documents, and notes current.

## 4. Track Work Activity

Use attendance records, leave requests, performance reviews, and training records to maintain a complete internal employment history.

## 5. Exit Management

When an employee exits, create an exit record and track:
- Resignation date
- Last working day
- Asset return status
- Final settlement status
- Knowledge transfer status
- Exit interview notes
