# Financial Controls Checklist

## Access Control

- Founder has complete finance access.
- Director has operational create, update, and view access.
- Viewer has read-only access.
- Archive actions are Founder-only.

## Journal Controls

- Voucher numbers must be unique.
- Debit and credit totals must match.
- Posted entries are immutable.
- Linked documents should be used for material entries when available.

## Banking Controls

- Bank account numbers should be masked.
- Reconciliation status should be updated after review.
- Bank transactions should be matched to journal entries when practical.

## Receivable Controls

- Outstanding invoice amounts must not exceed invoice totals.
- Overdue receivables should be reviewed weekly.
- Reminder status should be maintained.

## Payable Controls

- Vendor bills should be recorded before payment.
- Large or unusual expenses should use financial approvals.
- Overdue payables should be reviewed weekly.

## GST Controls

- GST records should be reviewed before filing.
- Input tax credit and output tax should be checked before finalizing.
- Filing status should be updated after submission.
